# AI Model Logos

Place the following logo files in this directory:

- `chatgpt.svg` (or `.png`) - ChatGPT logo
- `perplexity.svg` (or `.png`) - Perplexity logo  
- `gemini.svg` (or `.png`) - Gemini logo
- `deepseek.svg` (or `.png`) - Deepseek logo

The logos should be optimized for web use (SVG preferred, or PNG with transparent background).
